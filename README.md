# 📊 Customer Churn Prediction Platform

A unified cross-industry churn prediction system built with MLOps best practices.

## Features
- 🔮 Bulk churn predictions on CSV uploads (22,000+ customers)
- 🧍 Single customer lookup by ID with SHAP explanation
- 📊 Risk dashboard with industry breakdown
- 🏥 Model Health page with MLflow experiment tracking
- 🔗 Smart column mapping with fuzzy auto-matching

## Tech Stack
Python · Scikit-learn · XGBoost · MLflow · SHAP · Streamlit · Docker

## Industries Supported
E-Commerce · Banking · Telco
